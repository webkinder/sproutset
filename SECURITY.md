# Security Policy

## Supported versions

Sproutset is under active development. Security fixes are applied to the latest released version. Please make sure you are running the most recent release before reporting an issue.

## Reporting a vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, report them privately through either of the following channels:

- **Email:** [support@webkinder.ch](mailto:support@webkinder.ch)
- **GitHub:** [Private vulnerability reporting](https://github.com/webkinder/sproutset/security/advisories/new)

Please include a description of the vulnerability, steps to reproduce it, and the affected version. We will acknowledge your report as soon as possible and keep you informed of the progress toward a fix.
