# Sproutset

Modern responsive image management for projects using the Roots Acorn framework.

## Requirements

- PHP
- WordPress
- Roots Acorn

_Specific version requirements will be documented once the package is finalized._

## Installation

Install via Composer:

```bash
composer require webkinder/sproutset
```
